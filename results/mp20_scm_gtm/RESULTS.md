# MP-20 cartography — run results

Generated by `matmercator map` (from the committed feature cache) on the local MP-20 CSVs.
Every number below is **computed by this run** and reproducible from
`config.json` + `requirements.txt` with `random_state = 1234`; none is recalled
from memory. Raw metrics are in `report.json`; per-structure coordinates in
`gtm_coords.parquet`.

## Run configuration

| Item | Value |
|---|---|
| Dataset | MP-20 (train + val + test) |
| Structures featurized | 45,229 (27,136 / 9,047 / 9,046); 0 CIF parse failures |
| Descriptor | Sine Coulomb Matrix eigenvalues (matminer `eig`, unsorted), **20-dim** (= max cell size) |
| Standardization | z-score, scaler fit on frame set only |
| Frame set | 5,918 structures, stratified by space group, drawn from train |
| GTM (`ugtm.eGTM`) | k=16 (16×16 latent grid), m=4, s=0.3, regul=0.1, niter=200; posterior-mean projection |
| Unique space groups | 177 |
| Stack | pymatgen 2025.10.7, matminer 0.9.3, ugtm 2.3.0, scikit-learn 1.7.2, numpy 2.2.6, Python 3.10.12 |
| Wall time | GTM fit 15.4 s · project 0.45 s · plot 4.4 s · validate 7.2 s (local sandbox) |

The frame set came out at 5,918 rather than 6,000 because the stratified
sampler floors each occupied space group to ≥1 then allocates the remainder
proportionally (floor rounding undershoots slightly) — by design, so rare space
groups are represented.

## Map-quality metrics

Observed value vs. a label-shuffled null (200 permutations). For the continuous
properties the statistic is the between-cell fraction of variance ($\eta^2$) on a
20×20 binning of the GTM map; for crystal system it is mean k-NN label purity
(k=15) in map space.

| Quantity | Observed | Null (mean ± sd) | z | p |
|---|---|---|---|---|
| Band gap [eV] | 0.101 | 0.0079 ± 0.0007 | 129 | ≤ 0.005 |
| Formation energy [eV/atom] | 0.137 | 0.0080 ± 0.0006 | 217 | ≤ 0.005 |
| E above hull [eV/atom] | 0.059 | 0.0080 ± 0.0006 | 84 | ≤ 0.005 |
| Crystal-system k-NN purity | 0.291 | 0.167 ± 0.0005 | 228 | ≤ 0.005 |

`p ≤ 0.005` is the resolution floor of a 200-permutation test ($1/201$); the
observed value exceeded all 200 shuffles in every case.

## Reading these numbers

**Established (measured here).** Position on the SCM→GTM map carries
statistically unambiguous information about all four quantities — every observed
value is ~84–228 σ above its shuffled null. The map is not collapsed: points
fill the latent square (x,y std ≈ 0.43/0.50; 361/400 grid cells occupied;
densest cell holds 16% of points).

**Modest absolute organization.** The map captures only ~10–14% of band-gap and
formation-energy variance and ~6% of hull-distance variance. This is the honest,
expected ceiling for a *cheap, unsupervised* descriptor compressed to 2-D — it is
a calibration baseline, not a property predictor.

**Physically sensible ordering (inference).** $\eta^2$ ranks
formation energy (0.137) > band gap (0.101) > E_above_hull (0.059). SCM encodes
composition (atomic numbers) and periodic geometry, so a composition-dominated
quantity like formation energy is best organized; hull distance — a subtle
*relative* thermodynamic quantity — is least encoded. The ranking is what the
descriptor's content predicts.

**Metric calibration checks (established).**
- The continuous null mean (0.0080) matches the analytic ANOVA floor
  $(g-1)/(n-1) = 360/45228 = 0.0080$ for $g = 361$ occupied cells.
- The purity null (0.167) matches $\sum_i p_i^2$ over the crystal-system
  fractions (0.166) — the chance two random structures share a system.
  Both confirm the nulls are correct, not hand-set.

## Caveats (part of the result, not footnotes)

- **Unsupervised map, post-hoc coloring.** GTM never sees the properties;
  color-coding *validates* that structure-derived coordinates organize physics,
  nothing stronger. No property prediction is claimed.
- **Descriptor-bounded.** Conclusions are about the Sine Coulomb Matrix, the
  agreed cheap first pass. The Orbital Field Matrix (richer, 1024-dim) is the
  planned second pass to test whether gradients sharpen.
- **Properties are MP's own 0 K DFT (PBE) values** from the CSVs — PBE band gaps
  are systematically underestimated; energies are per-atom. Displayed, not
  recomputed.
- **Frame-set dependence.** Map *shape* depends on the 5,918-structure frame set
  (fixed here by the seed). A different frame set yields a rotated/relabelled but
  statistically comparable map.
- **GTM grid artifact.** The faint rectangular point-lattice in sparse map
  regions is the latent grid showing through where few structures project
  (posterior means snap toward grid nodes) — expected, not a bug.
- **Metric power scales with points-per-cell.** On small subsets the $\eta^2$
  null rises and power drops (a 750-structure check gave z≈0.3 for band gap).
  The full-set significance above is the meaningful result; don't read the metric
  on small slices.
- **Cache precision.** The staged feature cache is stored float32 (≤10⁻³ abs
  difference vs float64 on eigenvalues of magnitude 10²–10³); negligible after
  standardization. For bit-identical float64, run `matmercator run` directly.

## What this calibrates

The pipeline reproduces the feasibility study's recommended calibration: it runs
the full SCM→GTM path on the community-standard MP-20 set end-to-end in well
under a minute of model time, produces a non-degenerate map whose physical
organization is quantitatively significant and correctly ordered, and exposes
the exact knobs (descriptor, frame-set size/strata, GTM grid) needed before
scaling to Alexandria via the frame-set fit→project strategy.
